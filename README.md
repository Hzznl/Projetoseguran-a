Segurança do Trabalho App
Aplicação web desenvolvida com React, Vite e Tailwind CSS, voltada para o gerenciamento de informações e ações relacionadas à Segurança do Trabalho.
O sistema conta com funcionalidades interativas e modernas para facilitar a criação, visualização e controle de dados de segurança, integrando recursos visuais e relatórios em PDF.

✨ Funcionalidades
📊 Visualização de dados com gráficos dinâmicos usando Recharts

📝 Geração de relatórios em PDF com jsPDF e AutoTable

🎨 Interface responsiva e moderna com Tailwind CSS

🎯 Componentes acessíveis com Radix UI

🎥 Animações fluidas com Framer Motion

🔒 Estrutura modular e escalável

🛠 Tecnologias Utilizadas
React + Vite

Tailwind CSS + tailwind-merge + tailwindcss-animate

Radix UI (AlertDialog, Dialog, Dropdown, etc.)

Framer Motion para animações

Recharts para gráficos

jsPDF + AutoTable para relatórios

React Router DOM para navegação

Lucide React e Font Awesome para ícones
